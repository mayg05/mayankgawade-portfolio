# Mayank Gawade - Portfolio Website

A modern, responsive personal portfolio website showcasing my work as a Mechanical Engineering student.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Dark gradient theme with animated hero background
- **Smooth Animations**: Scroll-triggered animations and smooth transitions
- **Sections**: About, Education, Skills, Experience, Projects, Certifications, and Contact
- **Download Functionality**: Resume and certification downloads
- **Contact Form**: Client-side validated contact form

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Animations)
- Vanilla JavaScript
- No frameworks or external dependencies

## Deployment

This website is deployed on **GitHub Pages** for free. The site is accessible at:
`https://[username].github.io/MayankPortfolio`

For detailed deployment instructions, see [DEPLOYMENT_GUIDE.md](DEPLOYMENT_GUIDE.md)

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/[username]/[repository-name].git
   ```

2. Open `index.html` in your web browser, or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   ```

3. Navigate to `http://localhost:8000` in your browser
